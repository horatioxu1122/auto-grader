# HW3: Recursive Fibonacci

Write a Python program that computes the Nth Fibonacci number using **recursion**.

Requirements:
1. Your function must be named `fibonacci(n)`.
2. You must use recursion (no loops allowed for the core computation).
3. The program should handle `n = 0` (return 0) and `n = 1` (return 1) as base cases.
4. Include a brief docstring or comments explaining your approach.
5. When run as a script, it should print the first 10 Fibonacci numbers.
